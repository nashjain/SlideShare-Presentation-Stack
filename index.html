SlideShare Presentation Stack

A free responsive pure HTML5 widget for displaying SlideShare presentations

The Introduction

So in case you were wondering what this is all about ...

Presentation Stack is a free and responsive
HTML5 SlideShare widget

It's responsive, built on HTML5, JS and CSS3, and released for free under the MIT license, so use it for any of your personal or commercial projects!

The Details

Easy Plug and Play

Simply embed our code in your Website and you are ready to go.

Add this Code in your HTML and use your SlideShare username in the "data-username" field


<div id="slideshare_slides" data-username="slideshare-username"></div>
<script type="text/javascript" src="https://slideshare-presentation-stack.googlecode.com/git/slideshare.presentation.stack.js"></script>

<!-- Load Widget Asynchronously -->

<script type="text/javascript">
      slidesharePresentationStack.init();
</script>

That's all! Here's our sample widget.


Customize Your Widget

<div id="slideshare_slides" data-username="slideshare-username" data-num-presentations="10" data-slide-width="576" data-navigation-width="384" data-border="0px" data-border-color="#c4c4c4" data-navigation-slide-divider-thickness="1px" data-navigation-slide-divider-color="#FF2635" data-navigation-thumbnail-width-percentage="35%" data-vertical-scroll-bar-width="6px" data-vertical-scroll-bar-color="#FF2635" data-font-family="inherit" data-font-size="14px" data-font-weight="500" data-font-color="#333" data-slide-title-background="#fff" data-slide-title-background-on-hover="#F0F0F0" data-selected-slide-background-color-in-navigation="#fff" data-selected-slide-text-color-in-navigation="#333" data-selected-slide-text-font-weight-in-navigation="700"></div>
Data Property Name Required? Description
data-username Yes SlideShare User Name who's presentations you want to display
data-num-presentations No. Defaults to 20.
Max. presentations = 20		 Number of slides to be listed in the slide navigation sidebar.
data-slide-width 1 No. Defaults to 60% of the page width Width (in percentage) of the actual slide display window
data-navigation-width 2 No. Defaults to 40% of the page width Width (in percentage) of the slide navigator sidebar.
data-border 3 No. Defaults to 0px Width (in pixels) of the border around the widget.
data-border-color 3 No. Defaults to Grey (#c4c4c4) Color of the border around the widget.
data-navigation-slide-divider-thickness 4 No. Defaults to 1px Thickness (in pixels) of the divider between slides in the navigation sidebar.
data-navigation-slide-divider-color 4 No. Defaults to Red (#FF2635) Color of the divider between slides in the navigation sidebar.
data-navigation-thumbnail-width-percentage 5 No. Defaults to 35% of the sidebar width Width of the thumbnail in percentage of the sidebar width
data-vertical-scroll-bar-width 6 No. Defaults to 6px. Width (in pixels) of the scroll bar inside the navigation sidebar. (Works only in Chrome and Safari)
data-vertical-scroll-bar-color 6 No. Defaults to Red (#FF2635). Color of the scroll bar inside the navigation sidebar. (Works only in Chrome and Safari)
data-font-family 7 No. Defaults to the inherited font family. Font used to display the slide titles in the navigation sidebar.
Ex: data-font-family="Helvetica, Arial, sans-serif"
data-font-size 7 No. Defaults to 14px Size of the font used to display the slide titles in the navigation sidebar.
data-font-weight 7 No. Defaults to Normal (500) Weight of the font used to display the slide titles in the navigation sidebar. Allowed values: 300, 500 and 700
data-font-color 7 No. Defaults to Grey (#333) Color of the font used to display the slide titles in the navigation sidebar.
data-slide-title-background 7 No. Defaults to White (#fff) Background color of the area on which the slide titles are displayed in the navigation sidebar.
data-slide-title-background-on-hover 7 No. Defaults to Light Grey (#F0F0F0) Background color used to highlight the slide title when we hover over it in the navigation sidebar.
data-selected-slide-background-color-in-navigation 8 No. Defaults to White (#fff) Background color of the area on which the selected slide title is displayed in the navigation sidebar.
data-selected-slide-text-color-in-navigation 8 No. Defaults to Grey (#333) Color of the font used to display the selected slide title in the navigation sidebar.
data-selected-slide-text-font-weight-in-navigation 8 No. Defaults to Bold (700) Weight of the font used to display the selected slide title in the navigation sidebar. Allowed values: 300, 500 and 700
